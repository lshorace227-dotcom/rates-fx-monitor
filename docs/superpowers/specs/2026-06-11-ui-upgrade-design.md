# UI 全面升级设计（可视化 + 可交互）

- **日期**：2026-06-11
- **范围**：仅前端三件套（index.html / styles.css / app.js）。后端、数据层、build-snapshot、双模式检测逻辑全部不动；live 与 static 两模式同享全部新 UI；零新依赖。

## 1. 顶部状态条
标题右侧：模式徽章（`● 实时` 绿 / `◍ 每日快照` 蓝＋快照时间）＋「刷新」按钮（live 重拉数据；static 提示快照时间）。免责文字缩为小字。

## 2. 概览卡片（静态展示 → 可点交互）
- 点卡片 = 该标的上图（再点取消）；已上图卡片左边框亮蓝；与图表选择 chips 双向同步。
- 值颜色随 1M 变动红涨绿跌；sparkline 渐变面积＋末端点。
- 新增 52 周区间位置条（细横条＋游标点，(current−min)/(max−min)）。
- hover 抬升＋边框高亮；error 卡片不可点。

## 3. 图表区
- 预设对比视图按钮：美债收益率曲线（13W/5Y/10Y/30Y）、中美政策利率（SOFR vs LPR 1Y，自动指数化）、港美短端（SOFR vs HIBOR O/N）、人民币与美元（USD/CNY vs DXY，自动指数化）。
- 选择区改胶囊 chips（点击切换、选中高亮）。
- 十字准星 axisPointer cross；单序列渐变面积填充，多序列纯线。

## 4. 研判区（文本流 → 可视化）
- 每档三情景 → 一条三色堆叠概率条（红=上行/灰=中性/绿=下行，带百分比），细节用 `<details>` 折叠：第一档默认展开。
- 量化信号标尺：z 分数（−3~+3 游标）、区间位置（0–100% 游标）两个迷你可视化。
- 生成中 spinner。

## 5. 响应式
≤768px：卡片两列、chips 横向滚动、研判条全宽。桌面布局不变。

## 6. 验收
两模式浏览器实测：点卡上图、预设视图、区间条、概率堆叠条、信号标尺、移动宽度；合并 push 后线上复验。
